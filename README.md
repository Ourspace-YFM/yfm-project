# YourFleetMedia App

## Starting the App

### Start API Server
1. `$ cd yfm-project/api`
2. `$ yarn`
3. `$ yarn dev`

### Start Front-End
1. `$ cd yfm-project/react-web`
2. `$ yarn`
3. `$ yarn start`
